================================================================================
                        WHAT EACH SCRIPT DOES
================================================================================

run.py
------
Master script that:
‚úì Checks if dataset exists
‚úì Runs preprocessing automatically if needed
‚úì Loads NLP engine
‚úì Runs a quick test
‚úì Shows you what to do next

run_preprocessing.py
--------------------
Focused script that:
‚úì Checks for input CSV
‚úì Runs preprocessing only
‚úì Creates processed_tickets.csv
‚úì Shows statistics

preprocess_data.py
------------------
Core preprocessing:
‚úì Extracts 5 columns (Type, Subject, Description, Resolution, Priority)
‚úì Cleans text (removes emails, URLs, phone numbers, etc.)
‚úì Filters invalid data
‚úì Removes duplicates
‚úì Saves cleaned data

nlp_engine.py
-------------
NLP recommendation engine:
‚úì TF-IDF vectorization
‚úì Cosine similarity matching
‚úì Weighted feature combination
‚úì Top-K ranking
‚úì Resolution summarization
‚úì Model save/load

test_nlp.py
-----------
Interactive testing:
‚úì Load engine
‚úì Enter test tickets
‚úì See recommendations
‚úì Predefined test cases


================================================================================
                        TYPICAL WORKFLOW
================================================================================

First Time:
-----------
1. python run.py                    (automatic setup)
2. python test_nlp.py               (test with your tickets)
3. Read README.md                   (learn advanced features)

Daily Use:
----------
from nlp_engine import TicketNLPEngine

engine = TicketNLPEngine()
engine.load_data('processed_tickets.csv')

results = engine.find_similar_tickets(
    ticket_type="technical issue",
    subject="your subject",
    description="your description",
    top_k=3
)

================================================================================
                        TROUBLESHOOTING
================================================================================

Problem: FileNotFoundError
Solution: Run python run.py (it handles everything)

Problem: ModuleNotFoundError
Solution: pip install pandas numpy scikit-learn

Problem: No similar tickets found
Solution: Normal for very different tickets, try other test cases

Problem: Low similarity scores
Solution: Use more descriptive ticket descriptions
